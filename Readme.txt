X-CSL-Updater 0.1.1.4.beta
собрано для Mandriva|Ubuntu Linux на Qt 4.5.2
В папке sources исходники под Qt Creator 1.2.0
В папке release исполняемый файл релиза X-CSL-Updater
Релиз для Linux подготовил Saturn по всем вопросам обращаться на frizin@ya.ru

Как известно, для подключения к сети IVAO используется клиентская программа X-IvAp. В состав этой программы входит набор CSL-моделей объектов, который необходим для корректного отображения находящихся в сети других участников, на своих моделях самолетов, вертолетов, машин сопровождения и другой техники. Т.е. иными словами, эта библиотека работает на то, чтобы вы видели рядом (или в зоне видимости) других участников, которые подключились к сети на конкретных моделях самолетов (вертолетов), а также видеть эти модели в определенных ливреях (окрасках) разных авиакомпаний. Для симулятора MSFS(FSX) в сети IVAO такие задачи решает клиентское программное обеспечение сервиса MTL, основным преимуществом которого, является централизованное управление составом библиотек моделей самолётов (вертолетов) и ливрей многих авиакомпаний. Пользователь может с помощью клиентской программы загрузить по своему выбору (или целиком) эти библиотеки, которые потом будут работать в симуляторе. К сожалению, такого сервиса для X-Plane пока не существует. Но в глобальных планах развития этого проекта стоит именно реализация подобных задач в сети IVAO, для всего сообщества пользователей X-Plane. А пока наш проект будет реализовывать лишь часть этих задач.

Все, кто хоть раз сталкивался с вопросом поиска и установки CSL моделей, знают, насколько нестандартизованно и разобщённо появляются на различных порталах и сайтах подобные пакеты (или отдельные объекты) CSL-моделей, иногда дублируя друг друга, что приводит нередко к невозможности использовать симулятор в сетевых задачах. Иногда качество этих моделей также оставляет желать лучшего. И мы поставили перед собой первую задачу: начать централизованно собирать CSL-модели, тестировать, модернизировать при необходимости, и дать каждому пользователю возможность иметь этот комплект, при этом не утруждаясь вопросами установки и настройки. Главный критерий отбора - это востребованность конкретных моделей и работоспособность всего комплекта в целом.
Для таких первостепенных задач была создана  эта программа. 

Внимательно прочитайте указания по загрузке и установке!

Как было описано выше - задача состоит в организации централизованного пополнения и обновления пакета CSL-моделей у каждого пользователя, кто решил использовать наш проект. Исходя из этого, для правильной работы пакета следует удалить ВСЕ файлы находящиеся в папке \Resources\plugins\X-IvAp Resources\CSL симулятора. Если вы планируете пока ТОЛЬКО протестировать наш пакет, то сохраните всю папку CSL из плагина X-IvAp где-нибудь в стороне.

Краткая инструкция по запуску и работе программы.

1. Распаковать папку из архива на любой из ваших дисков.
2. Запустить программу:
* для Windows -> запустить файл X-CSL-Updater.exe в папке, которую вы распаковали из архива;
* для Linux -> запустить исполняемый файл X-CSL-Updater, находящийся в папке release;
* для Mac ->
3. В открывшемся окне программы нажать кнопку "Обзор" ("Browse") и указать месторасположение исполняемого файла X-Plane (Он находится прямо в папке, в которой установлен Ваш симулятор).
4. Нажать кнопку "Индексировать" ("Index"). В этот момент происходит сравнение установленных у вас CSL-моделей и существующих на сервере.
5. После завершения индексации, в строке "Статус" напротив каждого пакета моделей будет указано, требует ли данный пакет обновления или нет.
6. Для обновления пакетов, необходимо выделить соответствующую ему строку (если необходимо выделить несколько - удерживайте клавишу Ctrl) и затем нажмите кнопку "Обновить" ("Update").
