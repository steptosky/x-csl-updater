Добро пожаловать в проект X-CSL Package!

Для чего это сделано? В первую очередь, этот проект нацелен на всех пользователей авиасимулятора X-Plane, выполняющих полеты в сети IVAO®. Как известно, для подключения к сети IVAO используется клиентская программа X-IvAp. В состав этой программы входит набор CSL-моделей объектов, который необходим для корректного отображения находящихся в сети самолетов, вертолетов, машин сопровождения и другой техники других пользователей. Т.е. иными словами, эта библиотека работает на то, чтобы вы видели рядом (или в зоне видимости) других участников, которые подключились к сети на конкретных моделях самолетов (вертолетов), а также видеть эти модели в определенных ливреях (окрасках) разных авиакомпаний. Для симулятора MSFS(FSX), в сети IVAO, такие задачи решает сервис MTL, основным преимуществом которого, является единый состав моделей самолётов (вертолетов) и ливрей. Пользователь может с помощью клиентской программы MTL загрузить по своему выбору (или целиком) эти модели, которые потом будут работать в симуляторе. К сожалению, такого сервиса для X-Plane пока не существует. Но в глобальных планах развития этого проекта значится реализация этих задач в сети IVAO для всего сообщества пользователей X-Plane. А пока наш проект будет реализовывать лишь часть из них.

Все, кто хоть раз сталкивался с поиском и установкой CSL-моделей для X-Plane, знают насколько спонтанно и бессистемно появляются на различных порталах и сайтах подобные пакеты CSL-моделей, иногда дублируя друг друга, что приводит часто к неработоспособности как симулятора так и самих моделей. Иногда качество этих моделей также оставляет желать лучшего. Проект должен помочь решить первую задачу: начать централизованно собирать CSL-модели, тестировать, модернизировать при необходимости, собирать из них пакеты общий комплект моделей и ливрей. Второй задачей - дать каждому пользователю возможность иметь этот комплект, при этом не утруждаясь вопросами установки и настройки. Наш главный критерий отбора - это востребованность определённых моделей в сети работоспособность всего комплекта в целом.
Мы сделали клиентскую часть сервиса X-CSL под названием X-CSL-Updater.

Внимательно прочитайте указания по загрузке и установке!

Как было описано выше - задача состоит в организации централизованного пополнения и обновления пакета CSL-моделей у каждого пользователя, кто решил использовать наш проект. Исходя из этого, для правильной работы пакета следует удалить ВСЕ файлы находящиеся в папке \Resources\plugins\X-IvAp Resources\CSL симулятора. Если вы планируете пока ТОЛЬКО протестировать наш пакет, то сохраните всю папку CSL из плагина X-IvAp где-нибудь в стороне.

Краткая инструкция по запуску и работе программы.

1. Распаковать архива в любое место.
2. Запустить программу:
	* для Windows -> запустить программу X-CSL-Updater.exe;
	* для Linux -> запустить программу X-CSL-Updater;
	* для Mac -> смонтировать dmg образ с программой и запустить программу X-CSL-Updater.app
3. В открывшемся окне программы нажать кнопку "Обзор" ("Browse") и указать месторасположение исполняемого файла X-Plane (Он находится в корне папки, в которой установлен Ваш симулятор).
4. Нажать кнопку "Индексировать" ("Index"). В этот момент происходит сравнение установленных у вас CSL-моделей и существующих на сервере.
5. После завершения индексации, в строке "Статус" напротив каждого пакета моделей будет указано, требует ли данный пакет обновления или нет.
6. Для обновления пакетов, необходимо выделить соответствующую ему строку (если необходимо выделить несколько - удерживайте клавишу Ctrl или нажмите кнопку "Выделить Все") и затем нажмите кнопку "Обновить" ("Update").
