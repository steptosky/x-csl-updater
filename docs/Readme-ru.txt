Спасибо за то, что выбрали X-CSL Package!

Проект X-CSL-Package это централизованная библиотека 3d-моделей и текстур к ним для отображения траффика при полетах в сети IVAO в симуляторе X-Plane.
Проект разрабатывается, поддерживается и обслуживается командами X-Air and StepToSky Teams.
Библиотеку можно установить и постоянно обновлять с помощью нашего X-CSL-Updater.

- Библиотека X-CSL-Package разработана для использования в сети IVAO совместно с IVAO программным обеспечением (клиентами): X-IvAp и ALTITUDE.

- Данная версия X-CSL-Updater совместима Только с клиентом ALTITUDE. Для работы с X-IvAp скачайте более старую версию X-CSL-Updater (1.2.0 или более старую).

- Библиотека X-CSL-Package не совместима с любыми другими CSL библиотеками или отдельными CSL моделями. Пожалуйста очистите вашу папку с CSL моделями прежде чем использовать нашу библиотеку. Если вы собираетесь только попробовать нашу библиотеку, сохраните ваши CSL модели и библиотеки где-то в другом месте на вашем компьютере.

- Библиотека X-CSL-Package не совместима с любыми другими виртуальными авиационными сетями и их клиентами. Однако даже в этом случае вы можете попробовать использовать нашу библиотеку. Используйте Меню -> Установить путь для CSL(!) для установки кастомного пути для установки библиотеки. Если вы собираетесь только попробовать нашу библиотеку, сохраните ваши CSL модели и библиотеки где-то в другом месте на вашем компьютере.

Пожалуйста прочитайте инструкцию по загрузке и установке!

Краткая инструкция по запуску и работе программы.
1. Скачать архив (dmg-image for MacOS) в соответствии с вашей операционной системой.
1. Распаковать архив в любое место или смонтировать dmg-image и скопировать приложение X-CSL-Updater в папку Application если вы на MacOS.
2. Запустить программу:
	* для Windows -> запустить программу X-CSL-Updater.exe;
	* для Linux -> запустить скрипт X-CSL-Updater.sh;
	* для Mac -> запустить программу X-CSL-Updater.app
3. В открывшемся окне программы нажать кнопку "Обзор" и указать месторасположение исполняемого файла X-Plane (Он находится в корне папки, в которой установлен Ваш симулятор).
4. Нажать кнопку "Индексировать". В этот момент происходит сравнение установленных у вас CSL-моделей и существующих на сервере.
5. После завершения индексации, в строке "Status" напротив каждого пакета моделей будет указано, требует ли данный пакет обновления или нет.
6. Для обновления пакетов, необходимо выделить соответствующую ему строку (если необходимо выделить несколько - удерживайте клавишу Ctrl или нажмите кнопку "Выделить Все") и затем нажмите кнопку "Обновить".
